# 後龍好望角音樂祭會場導覽

這是一個可用 GitHub Pages 發布的手機導覽網頁，提供高鐵苗栗站、臺鐵豐富站往返後龍新港國中小的會場導覽。

## 使用方式

1. 用手機開啟 GitHub Pages 網址。
2. 選擇去程或回程。
3. 選擇高鐵苗栗站或臺鐵豐富站。
4. 按「開啟 Google Maps」取得即時導航。
5. 需要時可開啟相機，使用 AR 風格提示畫面。

## GitHub Pages 設定

Repository: `touchartfruit-arch/2026Haulong-Cape-of-Good-Hope-Music-Festival-Venue-Guide`

設定路徑：

1. 進入 GitHub repo 的 `Settings`
2. 選擇 `Pages`
3. Source 選 `Deploy from a branch`
4. Branch 選 `main`
5. Folder 選 `/root`
6. 儲存後等待 GitHub 產生網址

預期網址：

`https://touchartfruit-arch.github.io/2026Haulong-Cape-of-Good-Hope-Music-Festival-Venue-Guide/`

## 注意事項

相機功能需要 HTTPS。GitHub Pages 會提供 HTTPS，因此正式發布後手機較能正常使用相機權限。Facebook 內建瀏覽器可能限制相機；若遇到限制，請改用手機 Chrome 或 Safari 開啟。
